DSA签名

Posted on 2019-10-06

dsa签名

(p,q,g,y)是公钥

k共享攻击

r = (g**k mod p) mod q

s = (H(m) + xr) k**(-1) mod q

签名结果(r,s)。需要注意的是，这里与Elgamal很重要的不同是这里使用了哈希函数对消息进行了哈希处理。

s1 = (H(m1)+xr)k**-1 mod q

s2 = (H(m2)+xr)k**-1 mod q

因此，s1*k mod q = (H(m1) +xr) mod q

s2*k mod q = (H(m2)+ xr) mod q

k(s1 - s2) = H(m1) - H(m2) mod q

此时就可以解出k了，进一步求出私钥k。

循环移位

这段代码的意思是：

设32位数，4个字节

每个字节分别是x1,x2,x3,x4

返回值是x4 = x1 ^ x2 ^ x3 ^ x4

循环移位之后异或的组合并不变,所以就变成了8位的密钥。
